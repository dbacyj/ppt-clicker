📖 安装指引

  1. 安卓：下载 APK 后点击安装。若提示"未知来源"，前往 设置 → 安全 → 允许从此来源安装。
  2. Windows：下载 exe 双击运行。若被杀软拦截，请添加到信任列表（按键模拟的正常行为）。
  3. macOS：下载后 chmod +x 并运行。首次需在「系统设置 → 隐私 → 辅助功能」授权。

🔗 连接方式

蓝牙模式：手机开启蓝牙 → 电脑蓝牙搜索 "PPT Clicker" → 配对 → 翻页。

WiFi 模式：电脑运行伴侣程序 → 手机与电脑连同一 WiFi → App 自动发现 → 输入配对码。
