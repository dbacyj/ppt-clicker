PPT Clicker

Turn your phone into a presentation remote. Bluetooth HID or WiFi companion — no hardware required.

Download for Android
Latest release · Android 9+

Two ways to connect

Bluetooth HID

Your phone appears as a standard Bluetooth keyboard. Pair once, click forever — nothing to install on the PC.

Driver-free

WiFi Companion

Run a small companion program on the PC and control slides over your local network. Lower latency, slide-aware features ahead.

Optional

Features

  • Next / previous slide
  • Start presentation, exit fullscreen, black screen
  • Jump to first / last slide
  • Automatic Bluetooth pairing and reconnect
  • WiFi companion with auto-discovery and pairing code
  • Graceful Bluetooth → WiFi fallback on unsupported devices

On the roadmap: teleprompter with speaker notes, timer with haptic alerts, spotlight, voice control, AI assistant.

Getting started

Bluetooth mode (recommended)

  1. Install the app and enable Bluetooth
  2. On your PC, search Bluetooth devices and pair with "PPT Clicker"
  3. Start presenting — the app shows "Connected"

WiFi companion mode

  1. Run the companion program on your PC, note the pairing code
  2. Connect both devices to the same WiFi network
  3. In the app, select WiFi mode — it auto-discovers your PC, enter the code
