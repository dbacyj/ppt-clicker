📖 安装指引 / Install Guide

  1. Android：下载 APK 后点击安装。若提示"未知来源"，前往 设置 → 安全 → 允许从此来源安装。
    Download the APK and tap to install. Enable "Install unknown apps" if prompted.
  2. Windows：双击 exe 运行。若被杀软拦截，请添加到信任列表（按键模拟的正常行为）。
  3. macOS：chmod +x 后运行。首次需在「系统设置 → 隐私 → 辅助功能」授权。

🔗 连接方式 / How to Connect

蓝牙模式 / Bluetooth：手机开启蓝牙 → 电脑蓝牙搜索 "PPT Clicker" → 配对 → 翻页。

WiFi 模式：电脑运行伴侣程序 → 手机与电脑连同一 WiFi → App 自动发现 → 输入配对码。
