# 贡献指南

感谢你对 PPT Clicker 的关注！以下是参与贡献的基本流程。

## 如何贡献

1. **Fork** 本仓库
2. 创建功能分支：`git checkout -b feature/your-feature`
3. 提交改动：`git commit -m "feat: 描述你的改动"`
4. 推送分支：`git push origin feature/your-feature`
5. 在 GitHub 上创建 **Pull Request**

## 提交规范

- `feat:` 新功能
- `fix:` 修复 bug
- `docs:` 文档变更
- `refactor:` 重构
- `chore:` 构建/工具链变更

## 报告 Bug

请使用 GitHub Issues，附上：
- 设备型号和系统版本
- 复现步骤
- 期望行为与实际行为
- 相关日志（如有）

## 许可证

贡献的代码将采用 [MIT](LICENSE) 许可证。
